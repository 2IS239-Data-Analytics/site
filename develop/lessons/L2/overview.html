Lesson overviewÂ¶

During the second week we will dive into Python and learn some basic elements of Python programming language. We will also continue learning how to use the Course Environment and version control with Git + GitHub.

  1. Intro to version control and GitHub
  2. Basics of using Git
  3. Classroom for GitHub
  4. Some basic elements of Python
  5. Exercise 2: Writing scripts and using GitHub

Learning goalsÂ¶

After this weeks lesson your should be able to:

  • Understand the basic concepts of version control and why it is useful
  • Access your user space in Github.com and know how to use Git from terminal to clone repositories, commit changes, and push changes
  • Explain the concept of an index value
